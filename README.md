
# Flutter Api

built using php and laravel 

## Run Locally

Clone the project

```bash
  git clone https://git@github.com:WisdomOkechukwu/FlutterAPI.git
```

Go to the project directory

```bash
  cd my-project
```

Migrate the database

```bash
  php artisan migrate
```

Start the server

```bash
  Php artisan serve
```
